# Biblio-Tech

Sistema de gestión de biblioteca (proyecto académico UNAMBA) desarrollado en HTML, CSS y JavaScript.

## Estructura
- `index.html` — Login (admin / colaborador)
- `lector.html` — Identificación del lector
- `devoluciones.html` — Escaneo y devolución de libros
- `css/style.css` — Estilos generales
- `js/auth.js` — Lógica de login
- `js/lector.js` — Lógica de identificación del lector
- `js/devoluciones.js` — Lógica de escaneo y devolución

## Flujo de ramas (control de versiones)
- `master` — Rama principal. Valida y aprueba los merges.
- `main` — Módulo de Login.
- `test` — Módulo de Identificación del Lector.
